# WhatsApp Status Saver

A modern React Native application that allows users to view, download, share, and manage WhatsApp image and video statuses on Android devices.

---

## Features

- View WhatsApp image statuses
- View WhatsApp video statuses
- Download statuses
- Share statuses
- Saved Statuses section
- Full-screen media preview
- Search and filter
- Dark & Light Theme
- Automatic status refresh
- Offline support

---

## Technology Stack

- React Native
- JavaScript
- React Hooks
- React Navigation
- React Native FS
- React Native Video
- React Native Permissions
- AsyncStorage
